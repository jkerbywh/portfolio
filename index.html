Edge Detection and Image Smoothing

Applications like FaceTune and FaceApp allow their users to edit photograph of themselves or others, usually to touch up a feature to make the photo more appealing. I wrote a code in R that allows the user to take a .png file and smooth the image to the degree of their choosing. I plan to extend this project to allow the user to smooth the image in a specified area of the image. In addition, the code allows the user to detect edges in the photo and outline it (if they choose to) with a color of their choosing. This will be uploaded to GitHub shortly.


Games with Artificial Intelligence Opponent

Have you ever wanted to play a game like Tic-Tac-Toe or Connect Four, but you didn't have anyone to play with? This was the motivation for me to create a Tic-Tac-Toe and Connect Four game using RStudio. The code allows you to play against an AI opponent. The AI opponent for the Tic-Tac-Toe game is unbeatable. This project was adapted from a project that I completed in STAT 304 at UW-Madison. This will be uploaded to GitHub shortly.
